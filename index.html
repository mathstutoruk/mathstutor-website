Differentiation summary
Core rules and reminders used repeatedly across pure mathematics.

  • d/dx (sin x) = cos x
  • d/dx (cos x) = −sin x
  • d/dx (eˣ) = eˣ
  • product rule: (uv)' = u'v + uv'
  • quotient rule: (u/v)' = (u'v − uv') / v²
